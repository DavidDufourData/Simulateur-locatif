
# Renta Locative V6

Application React/Vite prête pour GitHub et Vercel.

## Installation

```bash
npm install
npm run dev
```

## Déploiement Vercel

- Framework Preset : Vite
- Build Command : `npm run build`
- Output Directory : `dist`

## Structure

```text
src/
  App.jsx
  main.jsx
  styles.css
api/
  verify-session.js
index.html
package.json
vite.config.js
vercel.json
```

## Fonctionnalités

- tableau de bord SaaS ;
- simulateur complet ;
- saisie décimale avec `,` ou `.` ;
- durée du prêt et durée de détention séparées ;
- portefeuille sauvegardé en localStorage ;
- comparateur de deux biens ;
- objectifs patrimoniaux ;
- graphiques patrimoniaux ;
- assistant automatique ;
- export PDF via impression ;
- mode sombre ;
- espace Premium.
